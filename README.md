# LINCID-Expectativas-SEM
Evaluación de expectativas estudiantiles con PLS-SEM en la Licenciatura en Ciencia de Datos de la UV
# Descripción del proyecto

Este repositorio contiene el cuestionario, los datos procesados, el código en R y los resultados del estudio realizado con la primera generación (2024) de la Licenciatura en Ingeniería en Ciencia de Datos (LINCID) de la Universidad Veracruzana (UV).

El objetivo principal fue analizar la estructura de las expectativas académicas, sociales e institucionales de los estudiantes de nuevo ingreso mediante un Modelo de Ecuaciones Estructurales basado en Mínimos Cuadrados Parciales (PLS-SEM).
